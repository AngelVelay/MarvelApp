# MARVEL APP

## Marvel App hecha en React utilizando React Router V6


***Como empezar*** 

>Clone the Repo
- `git clone https://github.com/AngelVelay/MarvelApp/`

> Install the dependencies you need  
- `npm install `

> Execute the command to view the app  
- `npm start`
